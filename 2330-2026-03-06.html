4. 產業分析與同業比較

4.1 ROE 分析與 Gordon Growth Model 辯證

ROE 基本計算：

  • 2025 年 ROE = 淨利 1,236 億 ÷ 股東權益 7,900 億 = 15.6%
  • 2025 年 EPS = NT$66.24 vs 每股淨值 (BPS) = 7,900 億 ÷ 259.3 億股 = NT$30.47
  • ROE = 66.24 ÷ 30.47 = 15.6% ✓ (驗證一致)

Gordon Growth Model 估值辯證：

理論 PB = (ROE - g) / (CoE - g)

  • Cost of Equity (CoE) = Rf + Beta × ERP
  • Rf (台灣 10 年公債) = 1.5%
  • Beta (半導體產業) = 1.25
  • ERP (市場風險溢酬) = 5.5%
  • CoE = 1.5% + 1.25 × 5.5% = 8.4%
  • 長期成長率 g = 4.5% (GDP 成長 + 台幣升值對沖)
  • 理論 PB = (15.6% - 4.5%) / (8.4% - 4.5%) = 11.1% / 3.9% = 2.85x
  • 實際 PB = 股價 1,890 ÷ BPS 30.47 = 6.20x
  • 溢價 = (6.20 - 2.85) / 2.85 = +117%

辯證結論：

台積電實際 PB (6.20x) 遠高於理論值 (2.85x)，溢價 117% 源於：

  • 護城河強度：全球獨家 CoWoS、3D IC 等專利技術，競爭對手 5-10 年內難以追趕
  • 市場地位：先進製程全球佔有率 > 54%，定價能力強於 ROE 15.6% 的靜態指標
  • 成長性溢價：未來 3-5 年預期 EPS CAGR 5-8%，高於 GDP 成長；但部分溢價已反映
  • 風險計價：地緣政治折價約 20-30%、匯率風險折價約 10%，抵消部分溢價

⚠️ 估值合理性判定：ROE 15.6% 與給定 Forward PE 28.5x 不匹配。 按「PEG 法則」(PE / 成長率)，若成長率 8%，合理 PE = 15.6% × 100 / 8% = 20-22x。 當前 28.5x Forward PE 隱含 EPS CAGR 應達 12%+，2026E 需驗證實現。
