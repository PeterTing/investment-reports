  • 低壓配電盤與開關設備：占營收約45%，毛利率28-32%

2.2 成長轉折論證（2025年EPS為何成長427%？）

驅動力詳解：

  1. 台電智慧電網專案（新增營收額外+30-40%）
    • 2024年6月起訂單開始落地，2025年全年交付額度預估占營收25-30%
    • 該專案毛利率預估30-32%，帶動整體毛利率提升至31.2%（vs 2024年25-27%）
  2. 再生能源併網需求（新增營收額外+15-20%）
    • 台灣2025年新增風電、太陽能容量超過3GW，需要配電與計量系統
    • 亞力為台灣再生能源並網的主要設備供應商
  3. 工業用電與資料中心需求（新增營收額外+10-15%）
    • 台灣製造業用電需求穩定成長，AI伺服器與資料中心用電暴增
    • 高毛利率的工業不斷電系統(UPS)與穩壓器銷量提升
  4. 毛利率改善（營利槓桿+200-300bp）
    • 原物料成本回降，加上產品組合改善（高毛利計量系統佔比上升）
    • 規模效應：營收成長導致營業費用率下降
⚠️ 增長可持續性警訊： 2025年EPS 2.8-3.0元乃多重利多疊加，2026年若各項訂單回歸常態，EPS成長將明顯放緩至6-20%。

3. 產業分析與同業比較

3.1 ROE 分析與估值辯證 (Gordon Growth Model)

ROE 基本計算：

  • 2025E EPS：2.8-3.0 元
  • 2025E 每股淨值 (BPS)：3.92 × (1 + 7.5% 留存配息率) ≈ 4.21 元
  • ROE = 2.9元 / 4.21元 = 68.9%
⚠️ ROE 異常高的解釋： 2025年EPS 2.9元乃高基數效應，分子大幅成長但分母(淨值)增長緩慢，導致ROE 68.9%遠高於長期可持續水準(15-20%)。此為一次性現象，不應以此ROE水準評估合理PE。

Gordon Growth Model 估值辯證：

  • Cost of Equity (CoE) = Rf + Beta × ERP = 1.3% + 1.0 × 5.5% = 6.8%
  • 長期成長率 g = 4% (保守，略高於GDP)
  • 理論 PB = (ROE - g) / (CoE - g) = (68.9% - 4%) / (6.8% - 4%) = 2,496%（極端失效）

結論： 由於2025年ROE異常高，Gordon Growth Model 無法適用。應改用正常年ROE水準(15-20%)進行計算。若採用正常ROE 16-17%：

  • 理論 PB = (16.5% - 4%) / (6.8% - 4%) = 4.375x
  • 目前實際 PB = 65 / 3.92 = 16.6x （遠高於理論值）
  • 結論：市場溢價原因主要來自2025-2026年短期高成長預期，而非長期價值

7.1 投資評級推導

評級機械規則：

  • 目標價 > 現價 × (1 + 15%) → 買進
  • 目標價在 現價 × (1 ± 15%) 區間 → 持有
  • 目標價 < 現價 × (1 - 15%) → 賣出

亞力電機評級計算：

  • 現價：NT$65.0
  • 目標價：NT$85.6
  • 上升空間：(85.6 - 65.0) / 65.0 = +31.7%
  • +15% 臨界點：NT$74.75
  • 結論：目標價(NT$85.6) > 現價 × 1.15 (NT$74.75) → 應判定為「買進」等級
