SCORE INTERPRETATION GUIDE
─────────────────────────────────────────────────────────────────────────────
50-60 ► 🟢 Strong Buy   (All dimensions aligned)
45-49 ► 🟡 Moderate Buy (Most dimensions positive)
40-44 ► 🟡 Hold        (Mixed signals)
35-39 ► 🔴 Weak Hold   (Caution warranted)

6 DIMENSIONS DEFINITION
─────────────────────────────────────────────────────────────────────────────
基本面:    Fundamental value, ROE, earnings, growth trajectory
籌碼面:    Institutional buying, insider accumulation, short squeeze
技術面:    Chart pattern, support/resistance, momentum indicators
消息面:    Catalyst events, earnings revisions, M&A rumors
總經:      Macro conditions, rate cycle, sector rotation
產業展望:  Industry trends, market share, competitive moat

RATING DEFINITIONS
─────────────────────────────────────────────────────────────────────────────
🟢 Buy:   Upside ≥15% with favorable fundamentals + technicals
🟡 Hold:  Fair value, wait for better entry or hold existing
🔴 Sell:  Overvalued or deteriorating fundamentals

COMMONLY ASKED QUESTIONS
─────────────────────────────────────────────────────────────────────────────
Q: Can I add more than 10 picks?
A: Yes, but design optimized for 10. Adjust CSS grid-template-columns if needed.

Q: How to add real-time data?
A: Use fetch() API to load JSON from data/ folder. See DEPLOYMENT.md

Q: Can I customize colors?
A: Yes, edit CSS hex codes in <style> section.

Q: Will it work offline?
A: Yes, completely offline-capable (no external dependencies).

Q: How to deploy to other platforms?
A: Works on Netlify, Vercel, AWS S3, Cloudflare Pages (just upload HTML files).

Q: How to add more daily dates?
A: Add entries to SAMPLE_PICKS object with date as key.

Q: Can I hide the archive section?
A: Yes, delete the entire archive-grid div from HTML.
